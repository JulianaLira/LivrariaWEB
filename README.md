# Instruções 

#Trocar na string de conexão de JULIANA\\SQLEXPRESS, para o seu usuário do banco.

#Fazer Migration

- update-database

#Executar Projeto
Ctrl + F5

- .Net 5.0
- Banco de Dados SQLServer 2012
- Bootstrap para FrontEnd
